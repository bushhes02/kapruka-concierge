# Kavi by Kapruka Demo Script

Use these prompts in order:

1. `birthday cake for mum under 6000`
2. `amma ta birthday cake ekak one 6000 aduwen`
3. `appa ku chocolate hamper venum 8000 kulla`
4. `flowers to Colombo tomorrow`
5. `track my order`

Expected behavior:

- Gemini understands the language and intent
- Kapruka MCP returns real products
- Product grouping stays deterministic
- Cart and checkout update inline
- Delivery validation runs before confirmation
- No order is created before the user clicks Confirm
- Tracking only runs when an order reference is provided

Optional judge prompts:

- `mal bouquet ekak balanna`
- `i want loonu`
- `i want perfume`
- `cart eka pennanna`
